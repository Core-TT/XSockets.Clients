## XSockets.Clients

This repository contains all clients for XSockets.NET 5.* that is maintained by Team XSockets Sweden AB

We currently have clients for:

- .NET 3.5
- .NET 4.5+
- UWP
- WP
- MonoTouch (iOS)
- MonoDroid (Android)
- .NET Microframework 4.2
- .NET Microframework 4.3
- JavaScript/TypeScript
- Arduino
